# Reddit Modmail Checker

A small tool that checks whether a subreddit has *any modmail conversations* using Reddit’s official API and OAuth2 authorization.

This utility lets subreddit moderators programmatically detect incoming modmail and build custom workflows around it (e.g., Discord notifications, dashboards, alerts, etc.).

## Features

- Authenticate with Reddit using OAuth2  
- Poll Reddit’s modmail API for your subreddit  
- Report whether there are new or unread modmail conversations  
- Light, simple, and useful for moderation automation

## Prerequisites

Before running this project you need:

- A Reddit account  
- Registered Reddit API credentials (client ID, client secret)  
- Moderator privileges on the subreddit you want to monitor  
- Node.js, Python, or your preferred runtime if not already installed
